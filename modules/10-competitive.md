## Quick Reference

- **Identify 5-10 benchmark accounts:** same/adjacent industry, mix of tiers (2-3 peers at 5K-50K, 2-3 aspirational at 50K-500K, 1-2 pattern sources at 500K+), min 3 posts/week
- **Extract per account (last 20-30 posts):** avg engagement, reaction type breakdown, post length, hook/closing styles, format distribution, hashtag strategy, posting frequency
- **Transferability filter matters:** personal stories + data-backed opinions + debate invitations work at any scale; product announcements + broad hashtags + high posting frequency do NOT work under 5K
- **Reaction type insight:** high Insightful % = teaching posts; high Empathy % = personal stories; high Like-only % = need stronger conversation triggers
- **Monthly refresh:** pull fresh data from 3-5 benchmarks, check for new patterns, check if you crossed a transferability threshold
- **Output:** Transferable Patterns Report at onboarding; "Competitive Update" section in monthly report
