# Module 06: Format & First Comment Rules

**Source of truth:** `../rulebook/linkedin-growth-engine-rulebook.md` — Section 5
**Load when:** Choosing post format, writing first comments

## Quick Reference

- **Format diversity (monthly):** Text + image = majority. Carousel = at least 2/month. Poll or multi-image = at least 1/month.
- **First comment is mandatory:** Published within 2 minutes of main post for every post.
- **First comment types:** Link drop (source reference), Extended context (backs a claim), Follow-up thought (adds depth), Soft CTA (max 2/week, "DMs open if..."), Hashtag placement (2-4 relevant hashtags).
- **First comment must add value.** Never restate the post.
- **If post references data or source, first comment MUST include the source link.**
- **Goal mapping:** Inbound leads = soft CTA. Thought leadership = extended context or link drop. Grow audience = follow-up thought.
- **Image rules:** Every text post should have a companion image (2x engagement). Consistent style across month. No stock-looking stock photos. AI images must not look AI-generated. Alt text mandatory.
- **Image anti-patterns:** Generic motivational quote cards, selfies with text overlay, screenshots of tweets, images unrelated to post content.

**Full rules:** For complete rules, read the Rulebook, Section 5.
